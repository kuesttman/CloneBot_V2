<h1 align="center"><b>CloneBot V2 🔥</b></h1>
<!---Introdução--->
<p><b>CloneBot V2 é inspirado no CloneBot do MsGsuite, que ficou desatualizado e com muitos erros. Nós dois o criamos para manter vivo o legado do CloneBot! O bot que ajudou milhares a clonar seus dados.❤️</b></p>
<p><b>1. O poderoso Telegram Bot baseado em Gclone para clonar facilmente os dados do Drive Compartilhado do Google Drive.⚡</b></p>
<p><b>2. Contas de serviço de uso do CloneBot V2 para clonar facilmente TBs de dados sem atingir o limite de upload/clone de 750 GB do Google Drive.♻️</b></p>
<p><b>3. É mais leve e executa apenas clonagem do lado do servidor para ter muito menos carga no sistema e não usar sua própria largura de banda.🗃️</b></p>
<p><b>4. Basta fornecer o link de compartilhamento de uma determinada unidade/pasta ou arquivo compartilhado e definir várias pastas de destino para clonar dados.🔗</b></p>

<h2><b>🫙Criando unidade compartilhada</b></h2>
<p><b>CloneBot V2 vem com a capacidade de clonar dados entre My Drive para Shared Drive ou Shared Drive para Shared Drive, mas em ambos os casos o Shared Drive é comum e necessário! Então, vamos ver como podemos criar nosso próprio Drive Compartilhado gratuitamente para usá-lo com o CloneBot V2.</b></p>
<h4><b>Primeiro visite o site <a href="https://msgsuite.eu.org">Shared Drive Generator</a> do MsGsuite ou você também pode usar o MsGsuite Shared Drive Generator <a href="https:/ /t.me/MSGuite_SD_Creator_Bot">Telegram Bot</a> para criar unidade compartilhada.</b></h4>
<h4><b><i>🌐Usando o site:</i></b></h4>
<h4><b>1. Abra o site e forneça os detalhes abaixo:</b></h4>
<p><b><code>Nome do Drive Compartilhado</code>: Digite o nome que você deseja dar ao seu Drive Compartilhado. Pode ser qualquer coisa, menos evitar o uso de Emojis para evitar erros UTF-8</b></p>
<p><b><code>Gmail ID</code>: insira o ID de e-mail da sua Conta do Google para o qual você deseja criar o Shared Drive.</b></p>
<p><b><code>Seleção de domínio</code>: usando a lista suspensa, selecione um domínio de trabalho através do qual você deseja criar a unidade compartilhada ou, se não tiver certeza, mantenha-o como <code>Aleatório< /código>.</b></p>
<img src="https://user-images.githubusercontent.com/87380104/180253709-a8c8f8b9-8f17-4837-adc8-6ecead61f42a.png">
<h4><b>2.Uma vez feito! Resolva o hcaptcha para provar que você é humano e clique em <code>CRIAR</code>.</b></h4>
<h4><b>3.Após a criação, adicione o Grupo do Google de suas contas de serviço nesse Drive compartilhado para usá-lo com o CloneBot V2. Não sabe como? <a href="https://github.com/TheCaduceus/CloneBot_V2#-adding-in-google-group">Aprenda aqui</a></b></h4>
<img src="https://user-images.githubusercontent.com/87380104/180262451-1240e3bd-b0f1-4403-837f-ee0b6c179beb.png">
<h4><b>4.Todos os problemas ou códigos de erro relacionados ao site da MsGsuite com suas soluções estão listados <a href="https://t.me/msgsuitechat/154243">aqui</a>.</b> </h4>
<h4><b><i>🤖Usando o Telegram Bot:</i></b></h4>
<h4><b>1.Abra o Telegram Bot do MsGsuite no Telegram e envie <code>/start</code>, depois clique em <code>CREATE TD</code>.</b></h4>
<h4><b>2. Agora forneça seu ID do Gmail para o qual deseja criar o Drive Compartilhado e dê um nome ao seu Drive Compartilhado.</b></h4>
<h4><b>3.Depois de fornecer os detalhes necessários! O bot solicitará que você selecione o domínio aleatoriamente ou manualmente! selecione de acordo com sua escolha. Finalmente! Você conseguiu.😘</b></h4>
<img src="https://user-images.githubusercontent.com/87380104/180258551-4bbd0802-0655-4ec1-97e8-32aa46dce184.png">
<p><b>⛔OBSERVAÇÃO: o Drive Compartilhado é um armazenamento temporário! use-o com cuidado e mantenha o backup de seus dados sempre com você.</b></p>

<h2><b>🕹️Implantação</b></h2>
<p><b>A implantação do CloneBot V2 é tão simples quanto seu uso! Existem muitos métodos listados abaixo para implantar o CloneBot facilmente, mas antes de implantá-lo, você precisa de alguns valores listados abaixo e como obtê-lo:</b></p>
<p><b>
	<code>path_to_gclone</code> - Caminho para o arquivo gclone, por padrão é <code>gclone</code> ou altere-o se estiver usando um diferente.<br><br>
	<code>telegram_token</code> - Obtenha o Token da API do Telegram do seu bot em <a href="https://t.me/BotFather">BotFather</a>.<br><br>
	<code>user_ids</code> - IDs de usuário do Telegram de usuários que podem usar seu CloneBot_V2. Separe-os usando <code>,</code> e o primeiro User ID é Admin.<br><br>
	<code>group_ids</code> - IDs de grupos do Telegram nos quais o CloneBot pode ser usado, caso contrário, mantenha <code>-1</code>. Separe-os usando <code>,</code><br><br>
	<code>gclone_para_override</code> - Deixe em branco se não souber o que é.
</b></p>
<p><b>⛔NOTA: Tudo em <code>config.ini</code> deve ser <code>Int</code>.</b></p>
